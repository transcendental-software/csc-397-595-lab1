# CSC 397/595 Lab Assignment 1 (SALU) Simple Arithmetical Logical Unit
**Jarvis College of Computing and Digital Media - DePaul University**

## Overview

In this lab assignment you will need to implement a Simple Artithmetical Logical Unit (SALU) that can perform logical and, logical or, addition and subtraction operations on 4-bit inputs.
You will need to write the implementation in Verilog and you may use the built-in gate primitives, including **and**, **nand**, **or**, **nor**, **xor**, **xnor** and **not**.
